Instalación a medida

Una solución diferente para cada negocio

No instalamos una aplicación estándar idéntica para todos. Elegimos únicamente las funciones que necesita el negocio.

Barbería

Puede incluir:

  • Clientes
  • Registro de visitas
  • Fidelización
  • Décimo servicio gratuito
  • Historial
  • Importes registrados

Dietista o centro de consultas

Puede incluir:

  • Clientes y bonos
  • Sesiones utilizadas y restantes
  • Registro de pagos
  • Reserva online con selección de servicio, día y hora
  • Consulta o cancelación de citas
  • Acceso privado del negocio

La configuración se adapta a cada empresa: no todas las instalaciones incluyen todas las funciones.
